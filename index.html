vbang2

bangs like ddg, but led by v

goto r hn w gh bd tb

syntax “v<bang> <keyword>” or “<keyword> v<bang>”.

e.g. “vg <keyword>” to google, “<keyword> vw” for wiki, etc.

api available at /search?q=%s for programmable calls.

see ddg’s catalog for more bangs (alphanums only).
